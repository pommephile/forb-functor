The Forbidden Pattern Functor on Peano Arithmetic

A Categorical Reformulation of Incompleteness

Shin UOTA
June 2026
No institutional affiliation
Companion paper to: The Forbidden Pattern Functor: Universal Constraint Extraction in Combinatorial Symbol Systems (2026). The present paper applies the Forb framework to the foundations of mathematics — Peano arithmetic and Gödel's incompleteness theorems — and introduces the Forb spectrum as a new invariant.
Abstract. We apply the Forbidden Pattern Functor Forb: PAextop → Set to the category of consistent extensions of Peano Arithmetic. Gödel's first incompleteness theorem is expressed as Forb(PA) ≠ ∅; Π₁-completeness is characterised by Forb(T) = ∅. We introduce three candidate invariants (provisional — see §5 for well-definedness caveats): the Forb spectrum Spec_Forb(T) — the set of complexity ranks at which true-but-unprovable Π₁ sentences appear; the Forb ordinal |T|Forb as a categorical analogue of the proof-theoretic ordinal; and the Forb density function fT(n). A computational exploration over 36 Π₁ sentences reveals that in our sample, all sentences at ranks with data (1–9, 11; rank 10 had no sentences generated in the sample) are fully decided by PA, with the first forbidden pattern appearing at rank 25 — the consistency statement Con(PA). Ranks 12–24 contain no sentences in the sample; the spectral gap boundary cannot be determined without denser generation. The minimal forbidden rank rmin(PA) is shown to be equivalent to the famous open problem of the simplest Π₁ sentence independent of PA. Connections to proof-theoretic ordinals, the MRDP theorem, and algorithmic information theory are discussed.

§1. Introduction

Gödel's incompleteness theorems (1931) stand among the deepest results in mathematical logic: any consistent, recursively axiomatisable theory containing a fragment of arithmetic cannot be both complete and consistent. Yet the standard formulation — the existence of a sentence G such that neither G nor ¬G is provable — treats incompleteness as a binary property. The present paper asks a different question: what is the structure of incompleteness?

We bring to bear the Forbidden Pattern Functor (Forb), introduced by UOTA (2026) for combinatorial symbol systems (where it takes values in BoolAlg; the codomain is corrected to Set for the arithmetic application — see §3) as a domain-neutral categorical tool for extracting structural constraints. Applied to the category of arithmetic theories, Forb maps each theory T to the set of Π₁ sentences that are true in the standard model ℕ but unprovable in T. The functorial structure frames incompleteness as a graded phenomenon: forbidden patterns appear only above a complexity threshold, and their distribution — the Forb spectrum — constitutes a new invariant.

§2. The Category PAext

Definition 2.1 (Category PAext)

Objects: Consistent, Σ₁-sound, recursively axiomatisable extensions T ⊇ PA in the language LPA = {0, S, +, ·, <}.

Morphisms: A unique morphism iT,S: T → S exists iff T ⊆ S (theory inclusion).

Composition: iS,U ∘ iT,S = iT,U. Identity: iT,T = idT.

PAext is a partially ordered set viewed as a thin category. It is small (countably many objects) and skeletal. PA is the initial object. Th(ℕ) — the complete theory of the standard model — is not an object (not recursively axiomatisable), so there is no terminal object. The category admits an ω-chain through iterated consistency extensions:

PA → PA + Con(PA) → PA + Con(PA) + Con(PA + Con(PA)) → …

§3. The Forb Functor on PAext

⚠ Correction (v3, post peer review, 2026-06-24): The original manuscript claimed Forb: PAextop → BoolAlg. This is incorrect: the candidate Boolean algebra ℬ = Π₁/_{ℕ⊧} collapses to the 2-element algebra since every Π₁ sentence is either true or false in ℕ. Moreover, Π₁ is not closed under negation. The honest codomain is Set.
Definition 3.1 (Forb: PAextop → Set)

On objects: Forb(T) = { [φ] : φ ∈ Π₁, ℕ ⊧ φ, T ⊬ φ }, where [φ] denotes the equivalence class of φ modulo logical equivalence in pure predicate calculus.

On morphisms: For iT,S: T → S (equivalently T ⊆ S), Forb(iT,S) = ι: Forb(S) ↪ Forb(T), the inclusion map.

Theorem 3.2 (Functoriality)

Forb is a contravariant functor from PAext to Set.

Proof. For iT,S: T → S and iS,U: S → U: any [φ] ∈ Forb(U) has U ⊬ φ; since T ⊆ S ⊆ U, T ⊬ φ, so [φ] ∈ Forb(T). The inclusion maps compose correctly: Forb(iT,U) = Forb(iT,S) ∘ Forb(iS,U). Identity preservation is trivial. ∎
Proposition 3.3 (Monotonicity)

T ⊆ S ⇒ Forb(S) ⊆ Forb(T). A stronger theory has fewer forbidden patterns.

§4. Gödel Incompleteness via Forb

Theorem 4.1 (Π₁-Completeness)

T is Π₁-complete ⇔ Forb(T) = ∅

Proof. (⇒) If T is Π₁-complete, every true Π₁ sentence is provable, so Forb(T) has no members. (⇐) If Forb(T) = ∅, every true Π₁ sentence is provable in T; T is Π₁-complete. ∎
Theorem 4.2 (Incompleteness — Forb form)

(i) [Gödel I, 1931] Forb(PA) ≠ ∅. The Gödel sentence G (a Π₁ sentence) satisfies G ∈ Forb(PA).

(ii) [Gödel II, 1931] Con(PA) ∈ Forb(PA). This requires the additional proof that PA ⊬ Con(PA), not merely the existence of an independent sentence.

Observation 4.3 (Gödel II in Forb)

While Gödel I guarantees Forb(PA) ≠ ∅ via the Gödel sentence G, Gödel II yields the sharper result that the specific Π₁ sentence Con(PA) — whose truth we accept on metatheoretic grounds — belongs to Forb(PA). Both are consequences of the same underlying mechanism (the fixpoint lemma applied to the proof predicate), but they are distinct results with distinct proofs.

Observation 4.4 (Reflection Hierarchy)

Let PA0 = PA, PAα+1 = PAα + Con(PAα), PAλ = ∪α<λ PAα. By Proposition 3.3 (Monotonicity):

Forb(PA0) ⊇ Forb(PA1) ⊇ Forb(PA2) ⊇ …

Turing (1939) and Feferman (1962) showed that along a Π₁-complete path through Kleene's O, the progression proves all true Π₁ sentences at the limit: the intersection ⋂α Forb(PAα) = ∅ along a Π₁-complete path through Kleene's O. (The limit notation limα→ω₁CK is an informal summary; the rigorous statement is that the set-theoretic intersection along the chosen path is empty.) This is a known result (Turing 1939, Feferman 1962); the Forb framework provides the language in which it is expressed.

§5. The Forb Spectrum and Novel Invariants

Definition 5.1 (Complexity Rank)

For a Π₁ sentence σ = ∀x₁…∀xk φ with φ a Δ₀ formula, define:

rank(σ) = k + d(φ) + c(φ)

where k = number of universal quantifiers, d(φ) = maximum nesting depth of function symbols, c(φ) = number of logical connectives.

For consistency statements, rank is dominated by the term complexity of the proof predicate Proof_PA(x,y).

⚠ Correction (v3, post peer review, 2026-06-24): An earlier version used "quantifier alternation depth" as a rank component. This is meaningless for Π₁ sentences (by definition, they contain no ∃ quantifiers). The corrected rank uses only universally meaningful measures.
Definition 5.2 (Forb Spectrum)

Spec_Forb(T) = { rank(φ) : [φ] ∈ Forb(T) }

The Forb spectrum is the set of complexity ranks at which T admits true-but-unprovable Π₁ sentences.

⚠ Well-definedness caveat (v3): Spec_Forb(T) is defined as {rank(φ) : [φ] ∈ Forb(T)}. Since rank(φ) (Definition 5.1) is not invariant under logical equivalence in pure predicate calculus (e.g., ∀x(x=x) and ∀x∀y(x=x) are equivalent but have different ranks), Spec_Forb(T) depends on the choice of representative from each equivalence class. Definitions 5.2–5.4 are therefore provisional pending a canonical representative selection rule or a rank function invariant under logical equivalence. See Limitation 9.
Definition 5.3 (Forb Ordinal)

|T|Forb = sup Spec_Forb(T) = sup{ rank(φ) : [φ] ∈ Forb(T) }

The Forb ordinal measures the complexity reach of T's incompleteness.

Definition 5.4 (Forb Density Function)

fT(n) = |{ [φ] ∈ Forb(T) : rank(φ) ≤ n }|

This counts the number of forbidden patterns up to complexity n. Its asymptotic behaviour fT(n) ∼ ? is a new computational problem.

Observation 5.5 (Spectral Gap — Computational)

In a sample of 36 Π₁ sentences (polynomial identities, inequalities, divisibility statements, and consistency assertions), all sentences with rank ≤ 9 and rank 11 are provable or refutable in PA. The first forbidden pattern found appears at rank 25 (assigned to Con(PA) under the Gödel numbering and rank convention of §5; see Limitation 3 for encoding-dependence) (Con(PA)), followed by rank 30 (Con(PA + Con(PA))).

This suggests rmin(PA) ≤ 25, where rmin(T) = min Spec_Forb(T).

Observation 5.6 (rmin as an Open Problem)

rmin(PA) — the minimal complexity of a Π₁ sentence independent of PA — is equivalent to the problem of the simplest Diophantine equation whose unsolvability is independent of PA (by MRDP). The best known bound is the universal Diophantine equation (≈9 variables, degree ≈10⁴⁵, Jones 1982). No natural Π₁ sentence independent of PA is known.

Observation 5.7 (AIT Connection)

The Forb density fT(n) is related to the density of random Π₁ truths (Chaitin's Ω in algorithmic information theory) and to Kolmogorov complexity thresholds. The framing as a Forb spectrum problem is new, but prior work in AIT and reverse mathematics provides partial results that constrain fT. Note: the current computational sample (36 sentences) is insufficient to estimate fPA(n); this is listed as OP6.

§6. Computational Exploration

§6.1. Methodology

36 Π₁ sentences were generated spanning polynomial identities (∀x(x+0=x)), commutative/associative laws, distributive identities, inequalities, divisibility statements, and consistency assertions. Truth in ℕ was determined manually for all sentences. Provability in PA was assessed using a combination of known results (PA proves commutative/additive/associative identities via induction; Con(PA) is independent) and, for simple polynomial identities, direct derivation from PA axioms.

§6.3. Interpretation

The spectral gap is the primary observation from this sample: PA decided all sampled Π₁ sentences of low complexity. Incompleteness manifests only above a complexity threshold. The exact location of this threshold — rmin(PA) — is one of the deepest open problems in proof theory. Between rank 12 and rank 24, no Π₁ sentences exist in our sample; a denser generation in this range would sharpen the gap boundary.

§7. Connection to Proof Theory

The proof-theoretic ordinal of PA is |PA| = ε₀ (Gentzen 1936). This ordinal is the supremum of ordinals whose well-foundedness PA can prove. The Forb ordinal |PA|Forb measures a different aspect: the complexity at which true-but-unprovable Π₁ sentences appear.

Observation 7.1 (Forb–Ordinal Relationship)

A natural question is whether |T|Forb — the supremum of finite ranks of independent Π₁ sentences — bears any systematic relationship to the proof-theoretic ordinal |T|. We note that no direct comparison is possible without extending the rank function to transfinite values: |T|Forb ∈ ℕ ∪ {ω} (since it is a supremum of finite integers), while |T| ranges over countable ordinals far larger than ω. For PA, |PA|Forb ≥ 30 (computational lower bound) while |PA| = ε₀. Whether a transfinite extension of the rank function can bridge this gap remains an open problem.

⚠ Correction (v3, post peer review, 2026-06-24): An earlier version conjectured |T|Forb ≈ ω|T|. This conjecture was type-incorrect: the left side is a natural number (or ω), while the right side is a transfinite ordinal for any theory with proof-theoretic ordinal ≥ 2. The conjecture has been withdrawn.

§8. Limitations

Limitations
  1. Set-valued, not BoolAlg-valued. The codomain is Set. No non-trivial Boolean algebra structure on Forb(T) is claimed. Prior work (UOTA 2026) used BoolAlg for combinatorial symbol systems where patterns form a natural BA; this does not carry over to Π₁ sentences.
  2. Small sample. The computational results are based on 36 sentences. A systematic generation covering thousands of Π₁ sentences is needed to draw robust conclusions about the spectral gap.
  3. Rank dependence on encoding. The rank of Con(PA) depends on the Gödel numbering of the proof predicate. Different reasonable numberings may shift the spectral gap. The existence of a gap is invariant; its numeric location is encoding-relative.
  4. rmin is open. We do not resolve the minimal independent Π₁ sentence problem — we merely identify its equivalence to rmin(PA).
  5. No AIT survey completed. The relationship between fT(n) and algorithmic information theory (Chaitin, Kolmogorov) requires a systematic literature survey to establish the precise novelty boundary.
  6. Conjectural ordinal relationship — withdrawn. An earlier version conjectured |T|Forb ≈ ω|T|. This was type-incorrect: |T|Forb ∈ ℕ ∪ {ω} (a sup of finite integers), while ω|T| is transfinite for any theory with proof-theoretic ordinal ≥ 2. The conjecture has been withdrawn. See Observation 7.1 for the corrected discussion.
  7. Ideal claim withdrawn. An earlier version claimed Forb(T) is an ideal in a Boolean algebra. This is false (counterexample: G ∈ Forb(PA), "0=0" ∉ Forb(PA), yet ℕ ⊧ "0=0"→G). The claim has been withdrawn.
  8. Data availability. The 36-sentence sample with full classification (formula, rank, truth value, provability status, and justification) and the computational framework are available from the author upon request. Public archival (Zenodo / GitHub) is planned prior to journal submission.
  9. Equivalence-class ambiguity. Spec_Forb(T) is defined as {rank(φ) : [φ] ∈ Forb(T)}, where [φ] is an equivalence class modulo logical equivalence in pure predicate calculus. Since the rank function (Definition 5.1) is not invariant under logical equivalence (e.g., ∀x(x=x) and ∀x∀y(x=x) are logically equivalent but have different ranks), the spectrum depends on the choice of representative from each equivalence class. A canonical representative selection rule, or a rank function refined to be invariant under logical equivalence, is needed for Spec_Forb to be well-defined.

§9. Open Problems

  1. OP1 — Minimal Forbidden Rank. Determine rmin(PA) exactly. Is it 25? Or are there simpler Π₁ sentences independent of PA waiting to be discovered? This is equivalent to the simplest Diophantine equation whose unsolvability is independent of PA.
  2. OP2 — Forb Density Asymptotics. Determine the asymptotic growth of fPA(n). Is it exponential? Related to Chaitin's Ω?
  3. OP3 — Forb–Ordinal Correspondence [WITHDRAWN]. The conjecture |T|Forb ≈ ω|T| was withdrawn as type-incorrect (|T|Forb ∈ ℕ∪{ω} while ω|T| is transfinite; see Observation 7.1 and Limitation 6). Open: can the rank function be extended to transfinite values such that a meaningful ordinal relationship with |T| holds?
  4. OP4 — Natural Π₁ Independent Statement. Find a "natural" (non-metamathematical) Π₁ sentence independent of PA. No such sentence is currently known. The Forb framework provides a systematic search methodology.
  5. OP5 — Forb and Forcing. Investigate the relationship between Forb and Boolean-valued models for set theory. The set-valued Forb functor may interact with forcing in Cohen's method.
  6. OP6 — Scaling the Computation. Scale the computational exploration to thousands of Π₁ sentences, using automated theorem provers for PA fragments and systematic generation of Diophantine Π₁ sentences.
