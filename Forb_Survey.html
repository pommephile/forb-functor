The Forbidden Pattern Functor

A Survey of Universal Constraint Extraction — From Chemistry to Categorical Unification

Shin UOTA
June 25, 2026

Abstract

The Forbidden Pattern Functor (Forb) extracts Boolean constraints from any combinatorial symbol system by enumerating patterns that never appear in a corpus. This survey synthesizes six recent results into a coherent picture: (1) the Cantor dimension DC of the forbidden set is transcendental if and only if the forbidden cardinality is not a power of two (OP10); (2) the boundary operator ∂ is a natural transformation in the category of surface-saturated CSS (OP11); (3) this yields a bifunctor B: SurfCSSop × Parabolicop → Trans unifying Forb with the Mandelbrot set; (4) Betti numbers of the allowed subgraph reveal complementary topological signatures — chemical grammar is 1D-rich (β₁-dominated) while linguistic grammar is 2D-rich (β₂-dominated) (OP3); (5) the de Bruijn Betti sequence determines the McNaughton–Papert locality horizon, placing English phonotactics at LT₃ (OP4); (6) the minimum feature dimension n is bounded below by the injectivity ceiling ⌈log₂|S|⌉, with sub-injective regimes possible only when forbidden patterns are F-separable (OP2). Together, these results establish the Forb functor as a universal mathematical structure connecting combinatorics, topology, number theory, and categorical dynamics.

1. Introduction

The Forbidden Pattern Functor [UOTA 2026a] maps any combinatorial symbol system (CSS) — a set of signs with binary feature encodings and an observed corpus — to the Boolean algebra of patterns that never appear. It is simultaneously a coalgebra (encoding unobserved transitions), a Dowker simplicial complex (encoding missing simplices), and a Formal Concept Analysis lattice (encoding empty-extent proper premises). These three formulations are provably equivalent [UOTA 2026a, Theorem 4.1].

This survey integrates six recent results that reveal the Forb functor as a gateway to deeper mathematics: transcendental number theory (§2), categorical naturality and the Mandelbrot connection (§3), topological grammar signatures (§4), locality hierarchies (§5), and information-theoretic bounds (§6). Each section is self-contained, but the cumulative picture is of a unified mathematical structure: the extraction of constraints from absence.

SURVEY SCOPE What this survey covers. This survey synthesizes OP2, OP3, OP4, OP10, and OP11 into a single narrative. Each result is presented with its theorem statement, proof sketch, and computational verification. Full proofs appear in the companion papers. 2. The Transcendental Dimension (OP10)

2.1 Cantor Dimension

DEFINITION Cantor Dimension. For a CSS with forbidden set F and effective dimension k (bits that vary within F):
DC(F) = log₂(|F|) / k

The Cantor dimension measures how the forbidden patterns fill the k-dimensional feature hypercube. For BOC/CLOC chemistry: |F| = 17, k = 5, giving DC ≈ 0.8175 — a number whose nature was an open question (OP10).

THEOREM DEDUCTIVE Theorem 1 (Transcendental Dichotomy — OP10). For any CSS with k > 0:
DC(F) is transcendental ⇔ |F| is not a power of 2
Equivalently: DC ∈ ℚ ∪ 𝒯. It is never algebraic irrational.

Proof sketch. (⇐) If DC were rational p/q, then |F| = 2kp/q would be a power of 2, contradiction. If DC were algebraic irrational, then by Gelfond–Schneider (1934), 2k·DC = |F| would be transcendental — but |F| ∈ ℕ is algebraic. Hence DC is transcendental. (⇒) |F| = 2m ⇒ DC = m/k ∈ ℚ. ∎

FINDING Genericity. Among the first 100 integers, 93 are not powers of 2. Transcendental DC is the generic case. The rational case (|F| a power of 2) has density zero in ℕ. Almost every CSS produces a transcendental Cantor dimension.

3. Categorical Unification with the Mandelbrot Set (OP11)

3.1 The Boundary Operator ∂

Define the boundary operator ∂ on bigrams: a forbidden bigram is SURFACE if it has an allowed Hamming-1 neighbor; otherwise INTERIOR. A CSS is surface-saturated if it has no INTERIOR patterns.

THEOREM DEDUCTIVE Theorem 2 (∂-Naturality). In the category SurfCSS where objects are surface-saturated CSS and morphisms are feature-refinement embeddings (append ghost dimensions), ∂ is a natural transformation. Verified exhaustively on 65,536 CSS instances with zero violations.

Proof sketch. Feature-refinement appends zero-bits to all feature vectors. A Hamming-1 flip in a ghost dimension produces a feature vector with no corresponding sign (all signs have zero in ghost bits). A flip in an original dimension maps to the same neighbor as before. Therefore the Hamming-1 neighbor set of each bigram is unchanged, and ∂-classification is preserved. ∎

THEOREM STRUCTURAL Theorem 3 (Bifunctor B). There exists a bifunctor B: SurfCSSop × Parabolicop → Trans whose projections extract DC (Forb side) and π (Mandelbrot side). In the category Emergence, the four-stage pipelines are isomorphic: Eforb(CSS) ≅ Emand(D). The isomorphism is structural — DC ≠ π in general, and should not be forced equal.

The bifunctor B maps objects to the transcendental number and morphisms to identities in the essentially discrete category Trans. ∎

4. Topological Grammar Signatures (OP3)

4.1 Complementary Betti Types

The allowed-subgraph flag complex of a CSS carries homology that characterizes the "shape" of its grammar. Computing Betti numbers for two real-world domains reveals a striking complementarity:

Domain|V||E|β₀β₁β₂Type
BOC/CLOC (chemistry)111362125201D-rich
English phonotactics39699107,1302D-rich
FINDING Complementary Topological Signatures. Chemical grammar (BOC) is 1D-rich: its complexity lies in independent cycles (β₁ = 252), with no higher structure (Q₇ is bipartite, so β₂ = 0). Linguistic grammar (English phonotactics) is 2D-rich: all 1-cycles are boundaries of 2-simplices (β₁ = 0), but 249 forbidden bigrams create 7,130 independent 2-dimensional voids (β₂ = 7,130). This 28.6× amplification — 16.4% forbidden edges generating 7,130 independent voids — reveals that phonotactic constraints are fundamentally triangular: pairs are allowed, but triples are forbidden.

4.2 Grammar Expressivity Metric

DEFINITION Grammar Expressivity.
G(CSS) = log₂(β₁ + 1) · (R / |S|)
where R is the number of proper premises (minimum Boolean constraints), |S| is the number of signs. G captures both topological complexity (β₁) and constraint density (R/|S|).

5.2 McNaughton–Papert Connection

Rather than the Chomsky hierarchy (all k-local constraints are regular), the de Bruijn Betti sequence determines position in the locally testable hierarchy LT₁ ⊂ LT₂ ⊂ LT₃ ⊂ … ⊂ Regular [McNaughton & Papert 1971]. The Forb functor, operating on k-grams, extracts LTk constraints. The locality horizon is the minimum k for which β₀(Gk) stabilizes or explodes.

6. Feature Dimension Lower Bound (OP2)

6.1 The Injectivity Ceiling

THEOREM DEDUCTIVE Theorem 4 (Feature Dimension Bound — OP2). For a CSS with sign set S, the minimum feature dimension n required to preserve all Boolean constraints satisfies:
  • n ≥ ⌈log₂(|S|)⌉ is sufficient (injective feature map always works);
  • n < ⌈log₂(|S|)⌉ is possible iff the forbidden set is F-separable — it can be distinguished from allowed patterns in a lower-dimensional feature space;
  • The information-theoretic lower bound is n ≥ ½·log₂(|S|² · H(∂)) where H(∂) is the entropy of the allowed/forbidden partition.

9. Limitations

  1. Survey scope. Proof sketches omit details present in companion papers. Full rigor for each theorem appears in the respective OP documents.
  2. Phonotactics data. English phonotactics results are based on CMUdict (134K words). Cross-linguistic validation across other languages has not been performed.
  3. Protein domains. The Pfam pipeline demonstration uses only 7 domains. Full-scale analysis requires the complete Pfam dataset (~20K domains).
  4. Mandelbrot side. The Parabolic category relies on Klebanoff (2001) for the π-emergence proof. Independent computational verification of the Mandelbrot-side pipeline has not been performed.
  5. Emergence category. The category Emergence (§3.2) is schematic. A fully rigorous construction requires axiomatizing the four-stage pipeline structure.
